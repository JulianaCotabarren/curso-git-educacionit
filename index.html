Working Directory

El directorio de trabajo corresponde a la carpeta donde inicializamos el repositorio de GIT. Muy pocas veces vamos a estar utilizando la linea de comandos ya que normalmente corresponde al area en donde se crean, editan y eliminan los archivos. Algunos de los comandos que podemos llegar a utilizar son los siguientes:

  • dir || ls: muestra el contenido de la carpeta
  • mkdir %lt;name>: crea una carpeta con el nombre especificado
  • cd <path>: abre la carpeta seleccionada
  • git init: inicializa un repositorio de git
  • rm -rf .git: elimina el repositorio de git
